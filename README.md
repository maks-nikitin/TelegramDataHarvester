# Telegram Data Harvester

---

## О проекте
**Telegram Data Harvester** — это программный прототип автономной системы для автоматизации сбора контента и анализа метрик вовлеченности (просмотры, реакции, репосты) в открытых Telegram-каналах. В отличие от простых ботов, система использует **протокол MTProto**, что позволяет извлекать глубокие метаданные без необходимости прав администратора, поверяя «интуицию» нейросети `mDeBERTa` жесткой логикой лингвистических фильтров.

## Ключевые возможности
*   **Инкрементальный сбор данных:** автоматическое определение новых публикаций и синхронизация метрик (глазки, лайки, репосты) за любой выбранный период.
*   **Zero-Shot Classification:** тематическая кластеризация постов «на лету» (Политика, Спорт, Экономика и др.) без предварительного обучения модели на конкретном канале.
*   **Advanced NLP Pipeline:** многоступенчатая очистка текста от технического шума и глубокая морфологическая лемматизация (начальная форма слов) для точного выделения трендов.
*   **Knowledge Caching:** хранение истории в реляционной базе данных `SQLite`, что исключает повторные ресурсоемкие вызовы ИИ для уже обработанных данных.
*   **Automated Reporting:** встроенный модуль генерации детализированных PDF-отчетов с визуализацией статистики и реестром сообщений.

##  Технологический стек
*   **Ядро:** Python 3.12, Telethon (MTProto)
*   **Искусственный интеллект:** Transformers (Hugging Face), PyTorch, Scikit-learn
*   **Лингвистика:** Pymorphy3, NLTK
*   **Данные:** SQLite, Pandas
*   **Интерфейс:** Streamlit, Plotly
*   **Отчетность:** FPDF2

## Как запустить проект

### 1. Клонирование репозитория
Откройте терминал и введите:
```bash
git clone https://github.com/maks-nikitin/TelegramDataHarvester.git
cd TelegramDataHarvester
```

### 2. Настройка окружения
1.Получите API_ID и API_HASH на сайте my.telegram.org.
2.Создайте файл .env в корневой директории и добавьте ваши данные:
```bash
API_ID=ваш_id
API_HASH=ваш_hash
```
3.Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

### 3. Первичная авторизация (First-time setup)
1.Перед запуском интерфейса необходимо создать файл зашифрованной сессии:
2.Запустите скрипт авторизации:
```bash
 python test_api.py
 ```
3.Введите ваш номер телефона и код подтверждения из Telegram в консоль.
4.После появления файла .session в папке data/ авторизация завершена.

### 4. Запуск приложения
```bash
streamlit run src/ui/app.py
```

Структура репозитория
* src/ui/app.py — реализация интерфейса дашборда и логики визуализации.
* src/parser/collector.py — модуль асинхронного сбора данных (MTProto Client).
* src/analytics/processor.py — ядро интеллектуального анализа (ИИ-классификация, NLP-обработка).
* src/analytics/report_generator.py — программная верстка аналитических PDF-отчетов.
* src/database/db_manager.py — менеджер управления персистентным хранилищем SQLite.
* config/categories.json — внешняя конфигурация тематических категорий и цветовой схемы UI.
